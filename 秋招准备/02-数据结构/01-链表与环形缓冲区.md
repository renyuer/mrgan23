## 链表 — 必能手写

### 单向链表
```c
typedef struct Node {
    int data;
    struct Node *next;
} Node;

// 创建节点
Node *create_node(int data) {
    Node *n = (Node *)malloc(sizeof(Node));
    if (n == NULL) return NULL;
    n->data = data;
    n->next = NULL;
    return n;
}

// 头插法
void insert_head(Node **head, int data) {
    Node *n = create_node(data);
    n->next = *head;
    *head = n;
}

// 删除节点（删除第一个值为 data 的节点）
void delete_node(Node **head, int data) {
    Node *curr = *head, *prev = NULL;
    while (curr != NULL && curr->data != data) {
        prev = curr;
        curr = curr->next;
    }
    if (curr == NULL) return;  // 没找到
    if (prev == NULL)
        *head = curr->next;    // 删头节点
    else
        prev->next = curr->next;
    free(curr);
}
```

### 反转链表（面试最高频）
```c
Node *reverse_list(Node *head) {
    Node *prev = NULL, *curr = head, *next;
    while (curr != NULL) {
        next = curr->next;   // 保存下一个
        curr->next = prev;   // 反转指针
        prev = curr;         // 前移
        curr = next;         // 前移
    }
    return prev;  // 新头
}
// 三指针法：prev/curr/next，走一步改一个指向
```

### 环形链表检测（快慢指针）
```c
int has_cycle(Node *head) {
    Node *slow = head, *fast = head;
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return 1;  // 追上了 → 有环
    }
    return 0;
}
```

## 环形缓冲区（嵌入式高频）

### 为什么嵌入式要它？
- 生产者-消费者模式（ISR 写，主循环读 — 你飞控里肯定用了）
- 无需动态分配，固定大小数组
- 无锁实现（单生产者单消费者时不需要锁）

### 实现
```c
#define BUF_SIZE 64

typedef struct {
    uint8_t buffer[BUF_SIZE];
    volatile uint16_t head;  // 写指针（ISR 可能修改，加 volatile）
    volatile uint16_t tail;  // 读指针
} ring_buf_t;

// 初始化
void rb_init(ring_buf_t *rb) {
    rb->head = 0;
    rb->tail = 0;
}

// 判空
int rb_is_empty(ring_buf_t *rb) {
    return rb->head == rb->tail;
}

// 判满（牺牲一个字节区分空/满）
int rb_is_full(ring_buf_t *rb) {
    return ((rb->head + 1) % BUF_SIZE) == rb->tail;
}

// 写入一个字节
int rb_put(ring_buf_t *rb, uint8_t data) {
    if (rb_is_full(rb)) return -1;  // 满，丢弃
    rb->buffer[rb->head] = data;
    rb->head = (rb->head + 1) % BUF_SIZE;
    return 0;
}

// 读取一个字节
int rb_get(ring_buf_t *rb, uint8_t *data) {
    if (rb_is_empty(rb)) return -1;  // 空
    *data = rb->buffer[rb->tail];
    rb->tail = (rb->tail + 1) % BUF_SIZE;
    return 0;
}

// 可用字节数
uint16_t rb_available(ring_buf_t *rb) {
    return (rb->head - rb->tail + BUF_SIZE) % BUF_SIZE;
}
```

### 关键问题
**Q: 为什么牺牲一个字节来区分空和满？**
A: 如果不牺牲，head==tail 时无法区分是空（刚初始化）还是满（写了一圈追上了）。牺牲一个字节后：head==tail 必为空，(head+1)%SIZE==tail 必为满。

**Q: 可以不用牺牲吗？**
A: 可以——额外维护一个 count 变量，push 时 count++，pop 时 count--。count==0 空，count==SIZE 满。
