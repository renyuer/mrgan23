## #define 宏定义——文本替换，不是函数调用

### 基本宏
```c
#define PI 3.14159
#define MAX_BUFFER 256
#define UART_BAUD 115200
```

### 宏函数——必须加括号！
```c
// 错误写法
#define SQUARE(x) x * x
SQUARE(1+2) → 1+2*1+2 = 5（不是9！）

// 正确写法
#define SQUARE(x) ((x) * (x))
SQUARE(1+2) → ((1+2) * (1+2)) = 9
```

### 面试必考：MIN/MAX 宏陷阱
```c
#define MIN(a, b) ((a) < (b) ? (a) : (b))

int x = 5;
int y = MIN(x++, 10);
// 展开：((x++) < (10) ? (x++) : (10))
// x 被加了两次！y = 6, x = 7
```

### 多行宏
```c
#define INIT_PERIPH() do {  \
    RCC_Init();             \
    GPIO_Init();            \
    Timer_Init();           \
} while(0)
// do-while(0) 技巧：保证宏在任何上下文中都是单条语句
```

## 条件编译——嵌入式必用

### 防止头文件重复包含
```c
#ifndef __MY_HEADER_H
#define __MY_HEADER_H

// 头文件内容
void init(void);

#endif
```

### 调试开关
```c
#define DEBUG  1

#if DEBUG
    #define LOG(fmt, ...) printf("[%s:%d] " fmt "\r\n", __FILE__, __LINE__, ##__VA_ARGS__)
#else
    #define LOG(fmt, ...)
#endif

// 发布时改 DEBUG 为 0，LOG 全部变空，不影响性能和代码体积
```

### 平台适配
```c
#if defined(STM32F103xE)
    #define FLASH_SIZE  512
#elif defined(STM32F103xC)
    #define FLASH_SIZE  256
#else
    #error "Unsupported MCU!"
#endif
```

### 面试常问：#ifdef 和 #if defined 的区别
- `#ifdef MACRO` 等价于 `#if defined(MACRO)`
- `#if defined` 更灵活：可以 `#if defined(A) && defined(B)` 组合多个条件

## # 和 ## 运算符

### # — 字符串化
```c
#define STR(x) #x
STR(hello) → "hello"
STR(123)   → "123"
```

### ## — 连接符
```c
#define CONCAT(a, b) a ## b
int CONCAT(var_, 1) = 10;  → int var_1 = 10;

// 嵌入式实战：外设初始化命名
#define GPIO_INIT(port) GPIO ## port ## _Init()
GPIO_INIT(A);  → GPIOA_Init();
GPIO_INIT(B);  → GPIOB_Init();
```

## 预定义宏（面试可能问）

| 宏 | 含义 |
|----|------|
| `__LINE__` | 当前行号 |
| `__FILE__` | 当前文件名 |
| `__DATE__` | 编译日期 |
| `__TIME__` | 编译时间 |
| `__STDC__` | 是否遵循 ANSI C |
| `__FUNCTION__` | 当前函数名（非标准但常用） |

## #error 和 #pragma

```c
// 编译期报错
#if MAX_TASKS > 32
    #error "Too many tasks!"
#endif

// pragma 常见用法
#pragma once              // 头文件只包含一次（替代 #ifndef）
#pragma pack(1)           // 1字节对齐
#pragma message("building...")  // 编译期输出信息
```

## inline 函数 vs 宏函数

```c
// 宏函数
#define MAX_MACRO(a, b) ((a) > (b) ? (a) : (b))

// 内联函数
static inline int max_inline(int a, int b) {
    return a > b ? a : b;
}
```

| | 宏函数 | inline 函数 |
|---|--------|-------------|
| 类型检查 | 无 | 有 |
| 调试 | 难 | 可设断点 |
| 副作用 | 可能（多次求值） | 无 |
| 展开 | 预处理器处理 | 编译器决定 |
